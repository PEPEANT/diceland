FAQ

런처에서 랭킹/채팅이 바로 보이지 않아요.
런처는 설치/실행/홍보용 페이지입니다. 실시간 기능은 별도 서버 연동이 필요해 지금은 정적 문구로만 제공되고, 실제 접속은 게임 안에서 진행됩니다.
Edge로 열기가 동작하지 않아요.
일부 인앱 브라우저에서는 microsoft-edge: 호출이 차단될 수 있습니다. 이런 경우 우측 상단 메뉴에서 "브라우저에서 열기"를 사용해 주세요.
